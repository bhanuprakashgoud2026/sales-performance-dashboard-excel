# Sales Performance Dashboard (Excel)
## Project Overview
This project presents an interactive Sales Performance Dashboard developed in Microsoft Excel to analyze business sales performance across regions, products, and employees.
The dashboard provides insights into:
- Total Sales
- Total Profit
- Total Orders
- Average Profit Margin
- Region-wise Sales Analysis
- Monthly Sales Trends
- Top 5 Products
- Employee Performance Analysis

## Business Insights
- West Region generated the highest sales.
- Profit margin remained close to 20%.
- Monthly sales peaked during mid-year.
- A small number of products contributed a major share of revenue.
- Employee performance varies significantly across the organization.
